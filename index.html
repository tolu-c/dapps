What is Dapps Reauthentication?

Dapps Reauthentication is an open source protocol for connecting decentralised applications to mobile wallets with QR code scanning or deep linking. A user can interact securely with any Dapp from their mobile phone, making Dapps Reauthentication wallets a safer choice compared to desktop or browser extension wallets.

How does it work?

Dapps Reauthentication connects web applications to supported mobile wallets Dapps Reauthentication session is started by a scanning a QR code (desktop) or by clicking an application deep link (mobile).
